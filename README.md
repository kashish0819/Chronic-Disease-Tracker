# 🏥 Chronic Disease Progression Prediction System

## 📌 Project Overview

The Chronic Disease Progression Prediction System is a Machine Learning project designed to predict the progression stage of chronic diseases using patient healthcare data.

The system analyzes factors such as age, disease type, medical history, lifestyle, biomarker score, medication dose, and heart rate to classify patients into different disease stages.

A Random Forest Classifier was used to build the prediction model, and a Streamlit web application was developed for real-time predictions.

---

## 🎯 Objectives

* Analyze chronic disease patient data.
* Perform data preprocessing and feature engineering.
* Build a machine learning model for disease stage prediction.
* Evaluate model performance.
* Develop a user-friendly web application.
